### Vue

**什么是Vue?**

```javascript
	1)是Vuejs是一套构建用户界面的框架，他只关注视图层的内容，他是前端的主流框架之一。
	2)前端框架主要负责的是MVC中的V这一层，主要工作就是和界面打交道，主要是用来对页面中的数据进行处理，以及制作前端页面相关的特效及动画.
	3)使用框架可以提高效率，使用Vuejs来做前端框架，对于处理数据的方面可以完全替换掉原有的dom操作的理念，通过vuejs框架提供的指令，前端开发人员不在关心dom是如何渲染的。
```

**Vue的基础语法：**

```javascript
var vm = new Vue({
 el: "#app",
        data: {
            "str1": "HellowWorld1",
            "str2": "HellowWorld2"
        }
})
```

**Vue的指令属性：**

```javascript
1）v-cloak
	使用v-cloak主要为了解决插值表达式的闪烁问题
	使用插值表达式的问题：
		在页面加载的过程中，在页面中的{{}}插值表达式首先会被页面认可为是html元素中的实在存在的内容，所以浏览器首先将{{}}展现在页面上，页面加载完成后，插值表达式{{}}才会真正的转变为动态赋予的值。这就会造成一种现象，如果我们将来终端在访问服务器的过程中，网速较慢那么我们的{{}}会首先展现出来，会一闪而过，出现插值表达式的闪烁问题。
2）v-text
3）v-html
4）插值表达式和v-text/v-html的区别
	a.对于元素中已经存在的值，只有插值表达式能够将原有的值保留并添加动态数据
	v-text和v-html之所以不能够保留元素标签对中原有的内容是因为，在使用以上两个指令属性之前会提前将标签对中的内容清空掉在赋予动态的值。
5）v-bind
	v-bind:是Vuejs中，提供绑定属性的指令
	对于v-bind在开发中一共有几种使用方式：
		a.直接使用指令属性v-bind来为元素中的属性进行绑定操作
		b.使用简化后的方式，将v-bind去除，直接使用:来对元素中的额属性进行绑定
		c.在使用v-bind对元素中的属性进行绑定的时候，可以直接在操作值的位置进行拼接
```

**使用v-on指令出发事件**

```javascript
1)v-on的基本使用：
	v-on:click="fun1"的形式来绑定事件
	相当于原生js中的onclick
    v-on:click=====@click
2)事件修饰符的使用:
	1.stop
    使用stop来对事件的冒泡机制进行阻止,js中的事件的冒泡是指，在触发了内层元素的同时，也会随之触发外侧元素（外层元素包裹了内层元素），在做点击的过程中，两个事件都会触发。
    2.prevent
    使用.prevent来阻止超连接默认的行为，所谓默认的行为指的是超链接中的href属性链接。
    3.capture
    4.self
```
